# Climate-Drivers Characterization & Flood Prediction — Tana River Basin

## Project Overview

This research characterizes flood dynamics in the Tana River Basin (Kenya) and develops prediction models to support early flood warnings. The project combines hydrological analysis, climate pattern attribution, satellite remote sensing, and machine learning.

---

## Key Research Objectives

• **Flood Event Characterization**: Identifies and analyzes flood events across three river monitoring stations (Bura, Galole, Garsen) using discharge thresholds and 14-day rolling maximum calculations

• **Seasonality Analysis**: Classifies floods by Kenya's bimodal rainfall pattern—Long Rains (Apr-Jun) and Short Rains (Oct-Dec)—to link climate patterns (El Niño, Indian Ocean Dipole) to river overflow timing and frequency

• **Temporal Trend Analysis**: Tracks decade-by-decade changes in flood frequency, duration, and intensity to understand historical patterns and long-term climate-driven changes

• **Spatial Flood Characterization**: Compares flood behavior across three stations to understand spatial propagation and correlations in peak discharge levels

• **Extreme Value Analysis**: Performs annual maximum and Peaks Over Threshold (POT) statistics for hydrological modeling and frequency analysis

• **Satellite Imagery Integration** *(In Development)*: Incorporating Sentinel-1 and Sentinel-2 imagery to create real-time flood extent maps and validate ground-based discharge measurements

• **ML Rainfall-to-River Prediction** *(In Development)*: Building machine learning models using CHIRPS rainfall data to forecast river level rises at multiple stations

• **Automated Flood Detection** *(In Development)*: Applying ML algorithms to satellite imagery to automatically classify flooded vs. dry areas for rapid flood mapping

• **Global Flood Model Validation** *(In Development)*: Testing GloFAS predictions against local observations to evaluate applicability for 20-day advance flood early warnings in the Tana Basin

---

## Data Sources

- **River discharge**: Daily readings from monitoring stations (Garissa, Bura, Galole/Hola, Garsen)
- **Geospatial**: Kenya administrative boundaries (GADM), water basins, hillshade elevation model
- **Future**: Sentinel-1/2 satellite imagery, CHIRPS rainfall data, GloFAS flood forecasts
